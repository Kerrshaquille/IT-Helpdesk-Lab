#Virtual Lab Setup Guide

## Lab Environment Overview
- Host PC: AMD Ryzen 7 5800X3D, 32GB RAM, RTX 3070 Ti
- Virtualization: Oracle VirtualBox
- Virtual Machines: 2 planned

## Virtual Machine 1: Windows 10 Client

**Purpose:** Simulate end-user computer for troubleshooting

**Specifications:**
- OS: Windows 10 Pro (64-bit)
- RAM: 4GB
- CPU: 2 cores
- Storage: 50GB
- Network: NAT

**Installation Steps:**
1. Created new VM in VirtualBox
2. Allocated 4GB RAM and 2 CPU cores
3. Attached Windows 10 ISO
4. Installed Windows 10 Pro
5. Created user account: helpdesk-user
