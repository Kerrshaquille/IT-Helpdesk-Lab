## Network Configuration Details

### Why Static IP?

- **Domain Controllers** require static IP addresses
- Clients use DC’s IP for DNS resolution
- Dynamic IP changes would break domain services
- Standard practice in production environments

### Host-Only Adapter Benefits

- Creates isolated network between VMs
- Doesn’t require physical network connection
- More secure for lab testing
- Simulates internal corporate network
